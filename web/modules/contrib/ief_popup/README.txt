INTRODUCTION
------------

The IEF popup module automatically transforms a complex IEF form
(insert/edit/remove/duplicate item) to a popup instead of an inline form.

Remove item #markup has been improved to make it more functional.

 * For a full description of the module visit:
   https://www.drupal.org/project/ief_popup

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/ief_popup


REQUIREMENTS
------------

 * Inline Entity Form (https://www.drupal.org/project/inline_entity_form)


INSTALLATION
------------

Install the IEF Popup module as you would normally install a contributed
Drupal module. Visit https://www.drupal.org/node/1897420 for further
information.


CONFIGURATION
-------------

1. Navigate to `Administration > Extend` and enable the module.
2. This module automatically transforms a complex IEF form
   (insert/edit item) to a popup instead of an inline form.
